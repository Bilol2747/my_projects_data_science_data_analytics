# Data Science & Data Analytics Projects

Ushbu repositoryda turli Data Science va Data Analytics loyihalari jamlangan. Loyihalar Python, SQL va Power BI yordamida tahlil qilingan va turli sohalarga tegishli masalalarni hal qiladi.

## Loyihalar ro‘yxati

1. **Bemorga mos keluvchi dori tavsiya qilish tizimi**  
   - Maqsad: bemorning sog‘lik holatiga mos dorini tavsiya qilish.  
   - Texnologiyalar: Python, Pandas, Scikit-learn, Matplotlib/Seaborn.  
   - [Notebook havolasi](https://github.com/Bilol2747/my_projects_data_science_data_analytics/blob/main/Bemorga_mos_keluvchi_dori_tavsiya_qilish_tizimi_loyihasi.ipynb)

2. **Diabet kasalligiga tashxis qo‘yish**  
   - Maqsad: bemorning parametrlariga qarab diabet tashxisini aniqlash.  
   - Texnologiyalar: Python, Pandas, Scikit-learn.  
   - [Notebook havolasi](https://github.com/Bilol2747/my_projects_data_science_data_analytics/blob/main/Diabet_kasalligiga_tashxis_qo'yish_loyihasi.ipynb)

3. **Ko’krak saratonini aniqlash**  
   - Maqsad: bemorning ma’lumotlari asosida ko’krak saratoni mavjudligini aniqlash.  
   - Texnologiyalar: Python, Pandas, Scikit-learn.  
   - [Notebook havolasi](https://github.com/Bilol2747/my_projects_data_science_data_analytics/blob/main/Ko'krak_saratonini_aniqlash_loyihasi.ipynb)

4. **Savdoni o‘sishini bashorat qilish**  
   - Maqsad: mahsulot yoki xizmat savdosining kelajakdagi o‘sishini bashorat qilish.  
   - Texnologiyalar: Python, Pandas, Scikit-learn.  
   - [Notebook havolasi](https://github.com/Bilol2747/my_projects_data_science_data_analytics/blob/main/Savdoni_o'sishini_bashorat_qilish.ipynb)

5. **Analytic texnik topshiriq**  
   - Maqsad: Data Analysis bo‘yicha texnik topshiriqni bajarish.  
   - Texnologiyalar: Python, Pandas, Matplotlib.  
   - [Notebook havolasi](https://github.com/Bilol2747/my_projects_data_science_data_analytics/blob/main/Analytic_texnik_topshiriq.ipynb)

## Umumiy texnologiyalar
- Python (Pandas, NumPy, Scikit-learn)
- SQL (ma’lumotlar bazasini tahlil qilish uchun)
- Power BI (dashboard va vizualizatsiya)
- Matplotlib va Seaborn (grafiklar va diagrammalar)

## Takomillashtirish imkoniyatlari
- Loyihalarni yanada real ma’lumotlar bilan kengaytirish
- Model va tavsiya tizimlarini yanada aniqlashtirish
- Vizualizatsiyalarni interaktiv qilish
